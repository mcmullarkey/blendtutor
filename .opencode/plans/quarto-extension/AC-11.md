## AC-11: blendtutor export-quarto CLI command — convert lesson YAML to .qmd fenced-div snippet

### Executable Spec
- **predicate:** 16 clauses. Positive structure (1-7): opens with ::: {.blendtutor language="<r|python>"} (parameterized, not hardcoded), prompt as prose, code_template as first code block (if Some), checks in {.checks} block (if non-empty), solution in {.solution} block (if Some), hints in {.hints} block (if Some), closes with :::. Negative (8-12): llm_evaluation_prompt ABSENT, gotchas ABSENT, no YAML keys leak, no empty blocks for absent fields, packages OMITTED. Adversarial guards (13-16): cross-fixture uniqueness (different lessons → different output), exit code 0 on valid / nonzero on invalid, backtick-in-template yields valid fence, prompt-contains-::: doesn't break div.
- **probe:** cargo test -p blendtutor-cli --test export_quarto (assert_cmd + predicates)
- **negative:** Emits llm_evaluation_prompt. Re-serializes YAML. Empty blocks for absent fields. Leaks gotchas. Hardcoded template (identical output for different lessons). Broken fence from backticks. ::: in prompt breaks div.
- **verification:** code (Rust integration test via assert_cmd)
- **fixture status:** existing lesson YAMLs (add_two_numbers.yaml, gotcha_lesson.yaml, python-course/add.yaml) + NEW export_quarto.rs test + NEW edge-case fixtures
- **rubric anchor:** §1.2, §1.3.1, §2.1, §2.2, §3.2, §5.1

### Design Intent
- §1: Commands enum gains ExportQuarto variant (8 subcommands total). Pure transform takes &Lesson, returns String. Illegal states unrepresentable at transform input.
- §2: Transform Lesson → String is pure (no I/O, deterministic). CLI command is thin effectful shell: read → parse → transform → stdout → ExitCode.
- §3: Pure transform in blendtutor-core. CLI handler in blendtutor-cli. Mirrors validate/run pattern.
- §4: commands::export_quarto — name says what (export to quarto), where (CLI), NOT (not import, not build site, not run).
- §5: Transform fn does one thing (render). Shell fn does one thing (wire I/O + exit code).
