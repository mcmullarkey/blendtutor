# API reference

The full `blendtutor-core` API reference is generated by rustdoc and published
alongside this book at [`/api/blendtutor_core/`](./api/blendtutor_core/index.html).

It is built with `RUSTDOCFLAGS="-D warnings"` and the crate declares
`#![deny(missing_docs)]`, so a broken intra-doc link or an undocumented public
item fails the build rather than shipping a gap in the reference.

> The `/api` path is assembled at deploy time by nesting the rustdoc output under
> the book. When reading this book from a local `mdbook serve`, that link only
> resolves after `scripts/check-docs.sh` (or the CI deploy) has merged the two.
