# Agent notes

Durable, dated notes on how this codebase fits together — written by
implementation slices so later slices don't re-derive the same context. One
file per topic. Append dated bullets; don't rewrite history.

Frontmatter template for a new page:

```
---
topic: <short-topic-slug>
created: <YYYY-MM-DD>
slices: [<issue numbers that touched this topic>]
---
```

Then dated bullets of the form `- YYYY-MM-DD (#issue): <what was learned>`.
Grep this directory before exploring the code for a topic.
