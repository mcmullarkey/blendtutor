# Introduction

`blendtutor` is a single-binary Rust CLI for authoring and running interactive
R and Python coding lessons with LLM feedback, and for building
browser-deployable lesson sites. It is the successor to the original R-package
`blendtutor`, rebuilt on a reusable `blendtutor-core` library so the same domain
logic can later back a Tauri GUI with no process boundary.

## Documentation stack

This site is built from two sources, merged into one deploy:

- **Narrative docs** — this book, written with [mdBook][mdbook].
- **API reference** — generated by [rustdoc][rustdoc] from `blendtutor-core` and
  nested under [`/api`](./api/blendtutor_core/index.html).

mdBook + rustdoc is the canonical Rust documentation stack: rustdoc owns the API
surface, mdBook owns the prose, and there is no Material-for-MkDocs equivalent
that subsumes both. Rather than fight that split, the two tools are each used for
what they are best at and stitched together into a single GitHub Pages artifact
at deploy time (see [Architecture](./architecture.md)).

[mdbook]: https://rust-lang.github.io/mdBook/
[rustdoc]: https://doc.rust-lang.org/rustdoc/
